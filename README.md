# 🌾 TerraPrice

**TerraPrice** is a smart system that predicts food prices based on **geographic location**. It uses real-world data from various regions to help farmers, researchers, and planners stay ahead of market trends.

---

## 🧠 What It Does

GeoGrain uses machine learning to forecast the prices of crops and food items by analyzing:

- 📍 Geographic data  
- 📈 Historical price trends  
- 🌦️ Optional weather patterns  
- 🗺️ Regional agricultural outputs

---

## 📌 Key Features

- 🔍 **Location-Based Predictions** — input a place, get smart price insights  
- 🧮 **Data-Driven Forecasting** — built with regression and time-series models  
- 🗂️ **Supports Multiple Crops** — easily adaptable for various commodities  
- 🖼️ **Interactive Map View** — visualize prices by area (coming soon!)
